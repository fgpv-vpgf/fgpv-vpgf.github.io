Class Panel

A panel represents a top level viewer panel (is an rv-panel node).

There are currently three panels:

  • main
  • side
  • table
example

Opening a panel

panel.open();

Note that panel refers to an instance of a panel object, which you'll see how to get from a PanelRegistry
