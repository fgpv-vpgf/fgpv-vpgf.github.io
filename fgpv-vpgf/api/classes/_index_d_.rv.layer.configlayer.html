state_changed: Event
  • Defined in index.d.ts:628

This event is fired when the layers state changes.

The state can be one of 'rv-error', 'rv-bad-projection', 'rv-loading', 'rv-refresh', and 'rv-loaded'. This event is always fired at least once with 'rv-loading' as the first state type.
