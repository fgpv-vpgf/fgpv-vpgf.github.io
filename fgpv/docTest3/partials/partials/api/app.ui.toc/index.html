app.ui.toc Module

The app.ui.toc module contains toc (or layer selector) panel.

directive

  • rvTocEntryFlag - The `rvTocEntryFlag` directive is one of the layer flags: type, data, out-of-scale, user-added. ```html ```
  • rvLayerItemSymbology - The `rvLayerItemSymbology` directive generates a symbology list and toggles its visibility. It also wiggles the stacked symbology icons on mouse over and focus. ```html ```
  • rvTocEntry - The `rvTocEntry` directive is a UI component for a layer or a layer group in the layer selector (toc). ```html ```
  • rvToc - The `rvToc` directive wraps and provides functionailty for the toc for the main panel.

service

  • tocService - The `tocService` service provides bindable layer data to the `TocController`'s template. __Lots of hardcoded sample config data.__
