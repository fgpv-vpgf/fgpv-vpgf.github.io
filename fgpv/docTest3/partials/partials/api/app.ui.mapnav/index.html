app.ui.mapnav Module

The app.ui.mapnav module contains the map navigation component.

directive

  • rvMapnavButton - The `rvMapnavButton` directive is a map navigation component button.
  • rvMapnav - The `rvMapnav` description handles the map navigation component.

service

  • mapNavigationService - The `mapNavigationService` service provides access to map navgiation compoent's actions like `zoom`, `geolocation`, `full extent` and `history extent`.
