app.ui.filters Module

The app.ui.filters module contains the filters and datatable panel.

directive

  • rvFiltersDefaultMenu - The `rvFiltersDefaultMenu` directive is a data panel menu allowing the user to change the layout of the data panel between 'full', 'default', and 'minimized', toggling search and filters controls and pringting or exporting data from the table.
  • rvFiltersDefault - The `rvFiltersDefault` directive is a filters and datatable panel component.
