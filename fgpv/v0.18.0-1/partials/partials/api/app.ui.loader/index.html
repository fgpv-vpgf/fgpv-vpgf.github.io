app.ui.loader Module

The app.ui.loader module contains components for layer import (file, online services, catalogue, etc.).

directive

  • rv-loader-file - The `rv-loader-file` directive creates a stepper interface for loading local files with geodata and turning them into layers.
  • rvLoaderMenu - The `rvLoaderMenu` directive description. TODO: add description
  • rv-loader-service - The `rv-loader-service` directive creates a stepper interface for importing map services and online-based geo files and turning them into layers.
