LayerBlueprint

 (service)

The LayerBlueprint service returns LayerBlueprint class which abstracts common elements of layer creating (either from file or online servcie). The LayerServiceBlueprint service returns LayerServiceBlueprint class to be used when creating layers from online services (supplied by config, RCS or user added). The LayerFileBlueprint service returns LayerFileBlueprint class to be used when creating layers from user-supplied files.
