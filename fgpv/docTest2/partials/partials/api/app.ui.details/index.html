app.ui.details Module

The app.ui.details module contains details pane for displaying point data and wms query results.

directive

  • rvDetailsContent - The `rvDetailsContent` directive renders the data content of details. To improve efficency a document fragment is first created prior to DOM insertion.
  • rvDetailsExpand - The `rvDetailsExpand` directive allows details to be expanded into a modal box when the expand button is clicked.
  • rvDetails - The `rvDetails` directive to display point data and wms query results. Where are multiple data items, displays a selector list on the left side, letting the user to select the item.
