app.ui.metadata Module

The app.ui.metadata module contains the metadata panel.

directive

  • rvMetadataContent - The `rvMetadataContent` directive renders the data content of metadata.
  • rvMetadataExpand - The `rvMetadataExpand` directive allows metadata to be expanded into a modal box when the expand button is clicked.
  • rvMetadataPanel - The `rvMetadataPanel` directive wraps the side panel's metadata content.
